Glosario

Claude para PFF · Todos los términos de la guía en un solo lugar

Reúne los términos definidos a lo largo de la guía. La etiqueta al final de cada término indica en qué página se explica con más detalle.

A
Archivo .skill Skills de PFF
Una skill empaquetada en un solo archivo (parecido a un .zip), lista para instalar. Por dentro contiene el SKILL.md y, según la tarea, carpetas de apoyo.
C
Changelog Skills de PFF
Archivo (CHANGELOG.md) que registra qué cambió en cada versión de una skill y cuándo, para saber qué incluye la versión vigente.
Claude Regular Introducción
El chat de siempre en claude.ai (web, móvil o escritorio). Funciona por turnos: tú escribes, Claude responde. Ideal para consultas rápidas, redacción y analizar un documento que le pegas o subes.
Conector / MCP Skills
Una conexión a una herramienta externa (Gmail, Google Drive, Calendar, Zoom) para que Claude consulte o actúe ahí. MCP es el estándar técnico detrás de los conectores.
Cowork Cowork
Modo "agente" de Claude en la app de escritorio. Ejecuta tareas de varios pasos por su cuenta, con acceso real a tus archivos, y genera entregables completos (Word, Excel, PDF, PowerPoint).
D
Distribución centralizada Skills de PFF
Un administrador publica una skill una sola vez en la cuenta de PFF y todo el equipo la recibe; las actualizaciones se propagan desde esa copia central. Editarla en una cuenta individual no afecta a las demás.
F
Fuente de verdad Skills de PFF
El único lugar oficial donde vive la versión correcta de algo (por ejemplo, una skill). Aunque circulen copias, solo una es la válida.
G
Google Drive for Desktop Cowork
Aplicación que monta Google Drive como una unidad local en tu computadora, para que Cowork pueda leer y escribir esos archivos directamente. También llamada File Stream.
I
Instrucciones de perfil Instructions for Claude Buenas prácticas
Texto que tú escribes y que se aplica a todos tus chats y sesiones. Solo tú puedes editarlo; la memoria de un compañero nunca toca tu perfil.
Instrucciones de proyecto Buenas prácticas
Contexto que aplica únicamente a los chats dentro de un proyecto específico. No se comparte con otros proyectos ni con chats sueltos.
M
Marketplace Skills de PFF
Un catálogo privado de plugins de la organización: un solo lugar central del que todos instalan los plugins aprobados, con control de versiones y actualización desde un único punto.
Memoria Buenas prácticas
Claude no recuerda automáticamente entre conversaciones: cada chat empieza en blanco. El contexto se conserva por tres vías: instrucciones de perfil, instrucciones de proyecto y memoria de proyectos en Cowork.
P
Plantilla Skills de PFF
Un documento base (por ejemplo, en Word) con el formato del despacho, que una skill usa para producir el resultado.
Plugin Skills
Un paquete que agrupa skills, conectores y herramientas, normalmente pensado para un rol o flujo de trabajo.
Prompt Introducción
La instrucción o pregunta que le escribes a Claude. Mientras más claro y específico el prompt, mejor la respuesta.
Proyecto Project Introducción
Un espacio que agrupa chats, archivos e instrucciones sobre un mismo tema o cliente. El contexto se reutiliza entre los chats del proyecto.
S
Skill (habilidad) Skills
Una capacidad especializada —un instructivo empaquetado, a veces con herramientas— que Claude carga solo cuando la tarea lo amerita (p. ej. crear un Word o un Excel).
SKILL.md / Markdown (.md) Skills de PFF
El archivo principal de instrucciones de una skill, escrito en Markdown: un formato de texto simple que abre cualquier editor.
T
Tarea programada Cowork
Una tarea que Claude ejecuta automáticamente en un horario definido (cada día, cada semana), disponible en Cowork. Se apoya en la skill schedule.
Token Cómo ahorrar tokens
La unidad con la que Claude mide el texto: aproximadamente ¾ de una palabra en español. Todo lo que entra y sale cuenta como tokens, y el plan tiene una cuota de uso.
V
Versión (MAYOR.MENOR) Skills de PFF
Numeración que indica qué tan reciente es una skill. El número MENOR sube por ajustes y reglas nuevas; el MAYOR, por cambios de fondo (nueva plantilla o cambio estructural).
