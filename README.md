# Kit Digital · PFF

Guías internas y herramientas digitales del despacho, publicadas con GitHub Pages.

## Estructura

- `index.html` — página de inicio con acceso a cada guía.
- `style.css` — estilos de la página de inicio.
- `pages/gqueues.html` — guía de GQueues.
- `pages/guia-claude-01-introduccion.html` a `04-glosario.html` — guía de Claude (4 partes).
- `pages/feedback.html` — formulario de feedback y updates.

## Publicar cambios

Cualquier cambio en `main` se refleja automáticamente en la página publicada (GitHub Pages), unos minutos después del push.
